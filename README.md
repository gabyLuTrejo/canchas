# Canchas

El siguiente programa a través de un menú te permite acceder a 3 diferentes canchas:
<li> Futbol </li>
<li> Basquet </li>
<li> Tenis </li>
<br>
Cada cancha se encuentra en una página diferente y se accede a ellas pormedio de enlaces.
